# SQL Case Studies

This repository contains multiple SQL case studies solved using **Microsoft SQL Server (MSSQL)**. Each case study focuses on analyzing different datasets and deriving meaningful insights using SQL queries.

## Case Studies

1. **[Restaurant Sales](case01_restaurant_sales)**  
   Analyze restaurant sales data to understand customer behavior, menu performance, and sales trends.

2. **[Movie Ratings](case02_movie_ratings)**  
   Explore movie ratings data to identify popular movies, rating patterns, and user preferences.

3. **[Pizza Delivery](case03_pizza_sales)**  
   Analyze pizza delivery operations and sales to derive business and delivery insights including customer ordering behavior, rider performance, pizza recipes/toppings analysis, cancellations, and time-based trends.

4. **[FoodFlix Streaming](case04_foodflix_streaming)**  
   Analyze online food delivery data to uncover customer preferences, ordering behavior, and satisfaction trends.

5. **[Cloud Bank](case05_cloudbank)**  
   Analyze banking data to understand customer transactions, account activities, and financial insights.

6. **[E-Commerce User Engagement](case06_e_commerce_user_engagement)**  
   Analyze user engagement to explore user behavior, sessions, campaigns, and conversion insights.

7. **[DMart EcoSales](case07_dmart_ecosales)**  
   Analyze retail sales data to evaluate the impact of sustainable packaging and business performance.

8. **[AdInsight Analytics](case08_adinsight_analytics)**  
   Analyze advertising data to measure campaign performance and marketing effectiveness.

9. **[Clothing Retail Insights](case09_clothing_retail_insights)**  
   Explore clothing sales data to understand customer preferences, product demand, and trends.

10. **[Chinook Music Store](case10_chinook_music_store)**  
   Gain insights into music sales, customer behavior, and employee performance.

11. **[Bike Rental](case11_bike_rental)**  
   Analyze bike rental data to identify usage trends and customer behavior.

12. **[E-Commerce Consumer Insights](case12_ecommerce_consumer_insights)**  
   Analyze ecommerce consumer behavior including demographics, purchase patterns, and satisfaction levels.
